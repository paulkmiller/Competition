The Competition


PadMapper

Tech

  • Sources data primarily from Craigslist, packages it—and data from other services—into an attractive format
  • Gets data from big management companies as well as individual landlords looking to list spare bedrooms
  • Built using nothing more than Bootstrap and jQuery + Google Maps

Design

  • The whole service is a top-down Google Map, and you can quickly jump between a location's Street View, WalkScore, and photos without leaving the site.

Special

  • Can set up alerts for new listings and that the data is updated in real-time, which has been the key to their success

Hotpads

Tech

  • Pulls listing information from sites like Craigslist + other major listing services w/ commercial properties to represent
  • Street view for every listing
  • Built using React.js
  • Can save favorite listings

Design

  • Interface is uncomplicated, smooth, and fast
  • Provides helpful guides to help renters make informed choices

Special

  • Specific maps for low-incoming housing or foreclosures
  • Allows users to contact properties and landlords right through the search map, while highlighting information like rental prices, photos ( including floorpans ), and more information once you click on a map marker ( property ). This is a very information dense website and packages it cleanly

Lovely

Tech

  • Another Craigslist-sourced, top-down map-based apartment search tool
  • Built using Angular & Moment.js
  • Stores application information after the first submission so applying for multiple properties is easier
  • Offers rent auto-payment if the apartment / housing community supports it.

Design

  • Organizes listings into clusters instead of inundating the map with a ton of markers; a smart idea toward keeping # of assets on the DOM at a minimum
  • Two kinds of markers— red & blue. Red represents listings that have been posted in the past few hours

Special

  • Aside from displaying all available information regarding a property once clicked, you can apply for rental directly through Lovely. Once the form has been filled, Lovely reaches out to the landlord and passes along the application information immediately after submission, giving Users a competitive edge.

Apartments.com

Tech

  • Offers immaculate 3D tours of listings with a combination of Photographs taken at various points within the listing and WebGL to model the area in the browser
  • Uses Bootstrap, jQuery, Google Maps, WebGL
  • Can easily and quickly add a listing with or without a user account

Design

  • Average apartment search website with clean UI, though based on poor choices. Obviously not designed for mobile

Special

  • Branded tours of the community / area of each listing, enforcing and ensuring quality as a brand trait

Redfin

Design

  • Redfin specific agents to do tours for a quick guide to the listing and the community; spark-notes for home searching

Tech

  • Can contact leasing agent directly through Redfin
  • Toll-free # to call for assistance + additional info ( Twillio )
  • Uses React, Webpack, and Node.js ( catch-22; everything, including large images, loads quickly but reacts very poorly when scrolling quickly )

Special

  • Branding, I guess? TBD
